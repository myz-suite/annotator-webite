# 隐私政策（Privacy Policy）

以下为**MyZ 高亮智能助理**与**MyZ 弹幕助手**的隐私说明，内容整理自仓库文档并面向终端用户。

**生效日期：2026-01-12**

**MyZ 高亮智能助理** 帮助你在浏览器中高亮、批注，并可选择启用 AI 对话、工具调用与向量检索等能力。**MyZ 弹幕助手** 则在 YouTube 页面抓取带时间码的评论并以弹幕形式本地呈现。除非你启用了可选网络功能，否则所有数据仅保存在本地浏览器中，我们不收集、不上传任何个人信息。

## 本地存储的数据
- 高亮文本、批注、标签、颜色与截图元数据
- 扩展设置（调色板、视图偏好、AI Provider 配置、模型能力与参数、MCP Server 配置、本地 tools 开关、向量检索配置、社区导入配置等）
- 你主动下载的截图（保存到你的设备）
- AI Provider / MCP Server 的连接信息（例如 Base URL、API Key、Headers、服务器地址等）仅保存在本地浏览器中

以上数据存储在浏览器本地空间（如扩展自带的本地存储），且仅与当前浏览器用户有关联，我们不会传输或备份至任何外部服务。**MyZ 高亮智能助理** / **MyZ 弹幕助手** 项目及维护者不会接触你的高亮数据、弹幕数据、队列条目或设置。

若你启用加密同步：

- 同步服务由你自行填写并采用第三方 S3 兼容对象存储（如 AWS S3、Cloudflare R2 或其他兼容服务）。
- 云端仅存储密文与必要元信息；同步密钥由你输入的密码派生，我们不保存你的密码，密钥仅保存在本地设备且不会上传。
- 任何由第三方存储服务引发的故障、计费或合规问题，应由你与对应服务提供商承担责任，我们不承担任何责任。

对 **MyZ 弹幕助手** 而言：

- 扩展会通过 YouTube 的私有接口读取当前视频的公开评论，以识别时间码。请求由你的浏览器直接发出，我们不代理、不保存、不窥视这些响应。
- 弹幕结果及其元信息（评论内容、作者、时间、点赞数等）会保存在浏览器本地，方便弹窗与扩展图标离线读取。
- 我们不收集任何用户信息或访问历史。所有解析结果都保存在你的本地设备中。

## 可选 AI Provider 与 MCP Server

当你在设置中启用 AI 相关功能时，你可能会配置：

- **自定义 AI Provider（OpenAI 兼容接口）**
- **MCP Server（外部 tools 服务）**

启用后，扩展会**由你的浏览器直接**向你所配置的服务器发起请求。发送内容可能包含你输入的 prompt、你选择附带的网页上下文、你添加的图片/截图、以及工具调用所需的参数等。

重要提示：

- 我们不运营任何用于接收、存储或处理这些请求的远程服务；所有网络请求均由你的浏览器直接发往你配置的第三方服务器。
- 你自定义的 AI Provider / MCP Server 的安全性、合规性、可用性、日志留存策略与收费策略均由该服务方决定；你需要自行评估并承担风险。
- 自定义 AI Provider 可能会消耗大量 token 并产生高额费用；你应自行了解计费规则并对费用负责。
- 请勿在未充分评估的情况下向第三方服务器发送隐私数据、敏感信息、受版权或受协议限制的内容。

## 端到端加密同步（E2EE）

所有同步到云端的数据均为端到端加密（E2EE），不会暴露任何明文或用户数据到同步平台。加密同步实现可公开审阅，代码位于 <https://github.com/myz-suite/sync/>。

## 可选 社区导入（X / Mastodon）

当你在设置中启用社区导入功能时，扩展会在你的浏览器中打开并访问 X（Twitter）或 Mastodon 的相关页面，并通过页面抓取的方式导入内容（无需调用官方 API）。

重要提示：

- 该功能是否可用、可导入的内容范围、以及导入质量会受到第三方网站页面结构、风控策略与条款变化影响。
- 你需要自行判断是否启用，并自行承担账号受限、访问受阻、抓取失败、内容不完整等风险与后果。
- 我们不运营任何远程服务；抓取与导入由你的浏览器直接与第三方网站交互，导入结果仅保存在本地浏览器中。

对于 **MyZ 弹幕助手**，通过 YouTube 的私有接口访问评论数据属于非公开渠道。我们不保证 Google 对此调用的容忍度，理论上 Google 可能对频繁调用的帐号或 IP 实施限流或阻断。若因此导致账号访问受限、视频无法加载或出现其他问题，责任由用户自行承担，我们无法提供补偿。

## 权限说明
- `activeTab`/`tabs`：**MyZ 高亮智能助理** 用于读取选区和截图；**MyZ 弹幕助手** 用于识别当前 YouTube 视频。
- `<all_urls>` 内容脚本：**MyZ 高亮智能助理** 用于浮动工具条与高亮；**MyZ 弹幕助手** 仅在 YouTube 页面运行，用于渲染弹幕。
- `storage`：在本地保存设置、弹幕缓存或高亮信息。
- `scripting`：**MyZ 弹幕助手** 用于在页面中展示弹幕浮层。
- YouTube 页面访问权限：**MyZ 弹幕助手** 需要读取公开评论，以便解析时间码并展示弹幕。

## 你的选择
- 通过 Dashboard 删除任何高亮、批注或截图。
- 随时禁用 AI Assistant / MCP tools / 社区导入 等可选网络功能。
- 卸载扩展后，浏览器会自动清理存储数据。

如需咨询或反馈，请通过[仓库的 Issue](https://github.com/myz-suite/myz-support/issues) 联系维护者。

## 我们不收集任何用户数据

**MyZ 高亮智能助理** 是纯客户端扩展，维护者没有任何后端服务用于接收、聚合或分析用户数据。扩展发出的全部网络请求只会发送到你配置或你正在访问的服务（例如你自定义的 AI Provider、MCP Server、以及 YouTube 等第三方站点）。启用这些可选网络功能即表示你理解并接受相关隐私风险，并自行承担由此产生的费用与责任。
