# 隐私政策（Privacy Policy）

以下为 MyZ Annotator 的隐私说明，内容整理自仓库文档并面向终端用户。

**生效日期：2025-10-24**

MyZ Annotator 帮助你在浏览器中高亮与批注网页内容。除非你启用可选的 LogSeq 同步，否则你的数据始终保存在本地浏览器中，我们不收集、不上传任何个人信息。

## 本地存储的数据
- 高亮文本、批注、标签、颜色与截图元数据
- 扩展设置（调色板、视图偏好、LogSeq 端点/页面/Token）
- 你主动下载的截图（保存到你的设备）

以上数据存储在 `chrome.storage.local`，仅与当前浏览器用户有关联，我们不会传输或备份至任何外部服务。

## 可选 LogSeq 同步
启用后，扩展将把你选择的高亮与批注发送到你配置的 LogSeq HTTP API（通常运行在 `http://127.0.0.1`）。数据仅在你的浏览器与本地 LogSeq 服务器之间传输，我们不运营服务器、不接收副本，也不记录这些请求。

同步负载包含：
- 转换为 Markdown 的内容
- 高亮 ID、创建时间与可选标签
- 你配置的认证 Token（若提供）用于与 LogSeq 通信

你可在设置中随时关闭同步，立刻停止所有网络请求。

## 权限说明
- `activeTab`/`tabs`：在你点击工具栏时读取当前选区并截图，不存储浏览记录。
- `<all_urls>` 内容脚本：用于在你选择的页面显示浮动工具条与渲染高亮，仅读取你选中的文本。
- `http://127.0.0.1/*` 与 `http://localhost/*`：用于与本地 LogSeq 通信（仅在启用同步时）。
- `storage`：在本地保存高亮与设置。

## 你的选择
- 通过 Dashboard 删除任何高亮、批注或截图。
- 随时禁用 LogSeq 同步。
- 卸载扩展后，浏览器会自动清理存储数据。
