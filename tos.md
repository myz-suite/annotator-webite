# 服务协议

**生效日期：2026-01-12**

本页面为 **MyZ 高亮智能助理** 与 **MyZ 弹幕助手** 浏览器扩展的中文服务协议摘要。安装或使用任一扩展即表示你同意以下条款。

## 使用条件
你需具备签订具有法律效力协议的能力。若你代表组织使用，需确认已获授权使该组织受本条款约束。

## 许可与合理使用
你获得以个人或内部业务目的安装并使用这些扩展的有限、非排他、可撤销许可。请勿在法律未允许的情况下对扩展进行反向工程、删除权利声明，或将扩展用于违法或侵权行为。

## 数据本地化
所有高亮、弹幕缓存、设置、认证信息等仅保存在你的浏览器本地。我们不提供任何用于接收、存储或处理用户数据的远程服务。若你启用加密同步，同步服务由你自行填写并采用第三方对象存储，云端仅保存密文与必要元信息；同步密钥由你输入的密码派生，我们不保存你的密码，密钥不会上传。

## 可选功能
- **自定义 AI Provider：** 你可以在设置中填写 OpenAI 兼容接口的 Base URL、API Key、请求头等信息，由浏览器直接向你填写的服务器发起请求。
- **MCP Server：** 你可以配置 MCP Server 地址与请求头，由浏览器直接向该服务器请求工具列表、发起工具调用等。
- **社区导入（X/Mastodon）：** 通过打开网页并抓取内容的方式导入数据（无需调用官方 API），可能需要你在浏览器中已登录对应账号。
- **加密同步（S3 兼容）：** 你可以配置自己的对象存储（如 AWS S3、Cloudflare R2 或其他 S3 兼容服务）并设置同步密钥，由浏览器直接与该存储交互。
- **MyZ 弹幕助手：** 浏览器调用 YouTube 的私有接口读取公开评论，以生成本地弹幕。我们不运行代理或服务器，你需自行遵循 YouTube/Google 的相关政策。

启用这些功能完全由你决定，并需自行承担使用第三方服务的风险与义务。尤其是：

- 你自定义的 AI Provider / MCP Server 由你自行选择与控制，其安全性、合规性、可用性、日志留存与收费策略均由该服务方决定；你需要自行评估并承担风险。
- 使用自定义 AI Provider 可能会消耗大量 token 并产生高额费用；你应自行了解计费规则并对费用负责。
- 社区导入功能会访问第三方网站并抓取内容，可能受该网站的条款、风控策略或接口变更影响；你需自行判断是否启用，并自行承担账号受限、访问受阻、内容不完整或其他后果。
- 请勿在未充分评估的情况下向第三方服务器发送隐私数据、敏感信息、受版权或受协议限制的内容。
- 我们不对你配置或使用第三方服务器（含自定义 AI Provider / MCP Server）所产生的任何损失、泄露、收费、争议或法律责任承担责任。
- 加密同步所使用的对象存储服务由你自行选择并承担责任；任何由第三方服务故障、数据丢失、计费争议或合规问题引发的后果，均由你与对应服务提供商承担，我们不承担任何责任。
- 我们同样不对你启用社区导入功能所产生的任何风险、争议或损失承担责任。

## 端到端加密同步说明
所有同步到云端的数据均为端到端加密（E2EE），不会暴露任何明文或用户数据到同步平台。若你遗忘同步密码，新的客户端将无法解密旧数据，需要重新全量加密并同步。加密同步实现可公开审阅，代码位于 <https://github.com/myz-suite/sync/>。
