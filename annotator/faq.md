# 常见问题（FAQ）

## 为什么安装后没有看到工具条？
请先选中页面中的文字，浮动工具条会在选区附近出现。若依然不可见，请在 `chrome://extensions/` 中重新加载扩展后重试。

## 标注会丢失吗？
数据保存在 `chrome.storage.local`。重新访问同一页面会自动恢复；若 DOM 变化导致无法定位，系统会提示你处理（可将其降级为纯文本保存）。

## 为什么有些网页片段会标注失败？
互联网的页面实现复杂多变，有些时候其结构上存在一些隐藏的元素，当我们选中以后无法处理就会出现这种情况。如果你遇到了这种问题，请尝试选中文字的时候少选开始或者结束的一两个字符，基本就能解决。如果问题依旧存在，请尝试在仓库[提交 Issue](https://github.com/myz-suite/myz-support/issues)，我们会对应分析该页面的问题给出解决方案。

## 如何导入/导出？
在 Dashboard 顶部菜单中可导出 JSON/Markdown；导入时支持「跳过 / 覆盖 / 保留最新」策略，并显示导入摘要。

## 什么是 Agent Skills？如何使用？
Agent Skills 是可复用的指令/流程包，会在对话前注入到模型提示中。你可以在 Dashboard → 设置 → AI 智能助理 → Agent Skills 中导入技能包，或在对话中使用 `@skill-name` 显式调用。

## 同步是否安全？需要我提供服务吗？
加密同步是端到端加密（E2EE），云端仅保存密文与必要元信息；你需要自行提供 S3 兼容对象存储（如 AWS S3、Cloudflare R2 或其他国内兼容服务）。同步密钥由你输入的密码派生，我们不保存你的密码；若遗忘密码，新客户端将无法解密旧数据，需要重新全量加密并同步。我们不提供线上服务，也不会接触你的明文数据。

## 如何反馈问题？
请在仓库[提交 Issue](https://github.com/myz-suite/myz-support/issues)，附带浏览器版本、页面链接、复现步骤与截图。
